# Selenium Automation Testing (Python)

## Deskripsi Project

Project ini merupakan implementasi automation testing menggunakan Selenium WebDriver dengan bahasa Python untuk menguji fitur Login pada web HowLongToBeat.

Automation dilakukan untuk memastikan fungsi login berjalan sesuai dengan requirement.

---

## Teknologi yang Digunakan

- Python
- Selenium WebDriver
- Google Chrome
- ChromeDriver

---

## Fitur yang Diuji

- Login dengan data valid
- Login dengan data invalid
- Validasi field kosong

---

## Cara Menjalankan Project

### Install Selenium
pip install selenium

### Jalankan Script
python main.py

### Masukkan nomor tes sesuai instruksi
Masukkan salah satu nomor tes lalu tekan enter
